CSS margin

The CSS margin property creates space outside an element's border, separating it from surrounding elements. You can:

  1. Set All Sides: Use margin: 20px; to apply the same margin on all four sides.
  2. Individual Sides: Use margin-top, margin-right, margin-bottom, and margin-left to control each side individually.
  3. Shorthand: Use margin: 10px 15px 20px 25px; (top, right, bottom, left).
  4. Auto Margin: Center elements horizontally with margin: 0 auto; if they have a width.
  5. Negative Margins: Use negative values to pull elements closer or overlap them.

CSS padding

The CSS padding property adds space inside an element, between its content and its border. It essentially creates "breathing room" within the element.

  1. Set All Sides: padding: 20px; applies the same padding on all four sides.
  2. Individual Sides: Use margin-top, margin-right, margin-bottom, and margin-left to control each side individually.
  3. Shorthand: Use margin: 10px 15px 20px 25px; (top, right, bottom, left).
  4. Auto Margin: Center elements horizontally with margin: 0 auto; if they have a width.
  5. Negative Margins: Use negative values to pull elements closer or overlap them.

CSS Box Model

The CSS Box Model is a foundational concept that defines how elements are structured and spaced on a web page. Each element in the Box Model consists of four parts, from innermost to outermost:

  1. Content: The actual text, image, or other content inside the element.
  2. Padding: Space between the content and the border. Padding pushes the content inward but doesn’t affect the element’s position relative to other elements.
  3. Border: The border surrounds the padding and content. It can have various styles (solid, dashed, etc.) and widths.
  4. Margin: The outermost space around the element, creating separation from neighboring elements.
