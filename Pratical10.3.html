CSS background property

The CSS background property controls how the background of an element appears.

  1. Background Colors:
    • You can set solid colours
    • Supports color names, hex codes, RGB, RGBA (with transparency), HSL
  2. BackgroundImages:
    • Can use any image URL
    • Multiple images supported
    • Images load on top of colors
  3. Repeat Settings:
    • Images can repeat horizontally, vertically, both, or not at all
    • Useful for patterns and textures
    • Default is to repeat both directions
  4. Position:
    • Place background anywhere in the element
    • Use keywords (top, bottom, center) or exact values
    • Control both horizontal and vertical position
  5. Size:
    • Cover: fills entire container, may crop
    • Contain: fits whole image, may leave gaps
    • Custom sizes using pixels or percentages
  6. Attachment:
    • Scroll: moves with content
    • Fixed: stays in place while scrolling
    • Local: scrolls within its container
  7. Shorthand Benefits:
    • Write everything in one line
    • More concise
    • Same functionality as individual properties
